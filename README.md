# Modul: Fallstudien zum objektorientierten Design

Sie finden hier den Code zur gleichnamigen Vorlesung mit diversen Beispielen. Die Vorlesung wird angeboten für Studierende der HNU in den Bachelor-Studiengängen [Digital Enterprise Management](https://www.hnu.de/dem), [Künstliche Intelligenz und Informationsmanagement](https://www.hnu.de/kim) sowie [Information Management Automotive](https://www.hnu.de/ima), jeweils im 2. Fachsemester.

Die dazugehörigen Vorlesungsmaterialien erhalten Sie entweder im entsprechenden [Moodle-Kurs](https://elearning.hnu.de) oder direkt auf [meiner internen Fachbereich-Website zu Software Engineering an der HNU (VPN notwendig)](https://se.hnu.de/fallstudien).

Viel Spaß beim Lernen einer der vielfältigsten und meistgenutzten Programmiersprachen der Welt!
